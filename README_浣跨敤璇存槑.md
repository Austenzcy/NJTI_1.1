# NJTI 第三版多语言网站

本版本基于第二版网站进行小幅更新：

1. 在原有中文、英文、法语、阿拉伯语、俄语基础上，新增西班牙语、日语、韩语。
2. 题目顺序已打乱，不再按四个维度模块连续展示；题目页仍不显示维度标签。
3. 继续保留第二版结果页结构：人格图片、结果文案、四维度百分比图表。
4. 继续使用第二版人格结果图片和第二版内容框架。

## 本地预览

解压后双击 `index.html` 即可打开。

## GitHub Pages 更新方式

将本文件夹内所有文件复制到你的 GitHub 本地仓库根目录，覆盖旧文件。然后用 GitHub Desktop：

1. Summary 填写：更新 NJTI 第三版
2. 点击 Commit to main
3. 点击 Push origin
4. 等待 GitHub Pages 自动部署

如果页面没有立刻更新，请等待 1-3 分钟，并按 `Ctrl + F5` 强制刷新。
